Bind
=======

### Installation
Just drop off bind.hpp somewhere in your includes.

### Usage
Compilation with C++11 enabled is the only requirement.
To enable MPI use -DBIND_MPI=[MPI threading level, e.g. MPI_THREAD_FUNNELED].
