bind
How?
Partitioned global workflow parallel programming model.

Sequential code — parallel execution.

Decorators
Parallelism is extracted using decorators that allow a “transactional DAG” collection.

bind::decorate<>(sort, a.begin(), a.end());

Partitioning
Scope guards place calls onto remote nodes. All transfers are implicit.

{ bind::node first(0); throw_coin(c); }
