bind
What keywords?
Prototype multi-node CPU/GPU applications fast.
One header, three keywords.
node
Bind execution scope to selected node.
{ bind::node first(0); throw_coin(c); }
cpu
Implicitely transfer args and bind the call to node's CPU.
bind::cpu(sort, a.begin(), a.end());
gpu
Implicitely transfer args and bind the call to node's CUDA device.

bind::gpu([factor](bind::array<float>& a){
    launch_scale_kernel(a.data(), factor);
}, a);
